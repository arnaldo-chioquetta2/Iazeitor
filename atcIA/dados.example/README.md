# Configuração local

Estes arquivos documentam o formato das configurações locais do Iazeitor.

Copie somente a estrutura necessária para a pasta `dados` criada ao lado do executável. Preencha chaves, credenciais e caminhos apenas nessa cópia local. A pasta `dados` é ignorada pelo Git e não deve ser versionada.

Os exemplos deste diretório não contêm chaves, senhas, endpoints privados ou caminhos pessoais.
