# Seguranca

## Politica provisoria

O Iazeitor ainda esta em desenvolvimento, entao a prioridade e evitar exposicao indevida de segredos e configuracoes locais.

## O que nao reportar publicamente

- tokens
- chaves de API
- senhas
- arquivos de configuracao locais
- logs com informacoes sensiveis

## Como reportar

Reporte vulnerabilidades de forma privada ao mantenedor. Nao abra issue publica para segredos ou falhas que exponham credenciais.

## Contato

Preencher pelo mantenedor do projeto.
